          Hello, I'm stuck on this IGMP issue.  I am trying to replace my ISP generic modem/router with the ER8 (OS ver 1.5.0). My setup with the ISP's router works, but not with ER8. Here is my setup so far. Hopefully, someone can spot out something in my router ER8 config.  Thanks

           

          1. I have eth1 as my internet port (WAN DHCP IP from the ISP), -- Note: (I'm using 2.2.2.2 as a generic IP and not my real ISP's IP for easy demo)

           

          2. I have eth2 config for my 5 Public IPs routing to eth1. -- Note: (I'm using 3.3.3.0/29 as a generic IP and not my real ISP's IP for easy demo)

           

          3. I dedicated a router for Video only. The video router has multicast enable and already working, The Video router is using one of the static IPs, connecting to er8 (eth2),  and the set top boxes are behinh the Video router.
